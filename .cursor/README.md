# Configuração MCP do Supabase

Este diretório contém configurações relacionadas ao Cursor IDE.

## Configuração do MCP

Para configurar o MCP do Supabase no Cursor, siga as instruções no arquivo `MCP_SETUP.md` na raiz do projeto.

## Localização das Configurações do Cursor

As configurações do Cursor geralmente ficam em:
- **Windows**: `%APPDATA%\Cursor\User\settings.json`
- **Mac**: `~/Library/Application Support/Cursor/User/settings.json`
- **Linux**: `~/.config/Cursor/User/settings.json`

## Nota

O arquivo `mcp.json` (se necessário) pode precisar ser criado manualmente nas configurações do Cursor, pois arquivos de configuração do editor geralmente não são versionados no repositório.
